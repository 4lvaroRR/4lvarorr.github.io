Zona Maker

¿Para que se usa la zona maker?

Esta zona de diseño esta especificamente diseñada y equipado con ordenadores y elementos específicos para llevar a cabo tareas específicas de diseño.Se pueden prácticar diferentes tipos de diseño que se pueden prácticar en este aula:

  • 1. Ensamblaje y reparación de ordenadores : Utiliza las mesas para armar y reparar computadoras personalizadas.

  • 2. Laboratorio de pruebas : Configura un área para probar configuraciones de hardware y software.

  • 3. Centro de datos local : Usa el rack para alojar servidores locales.

  • 4. Espacio de coworking : Ofrece el espacio como un área de trabajo compartido para profesionales de la tecnología.

  • 5. Centro de capacitación : Utiliza las mesas para impartir cursos de montaje de ordenadores y mantenimiento.

En resumen, esta zona maker se adapta a diversas disciplinas que involucran la creatividad y el uso de software especializado. Proporciona un entorno equipado con las herramientas necesarias para facilitar el proceso de desmontaje y montaje, ya sea gráfico, industrial, de hardware o cualquier otra disciplina relacionada con la computación.
